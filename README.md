# rotate-slider

Проект собран с помощью Webpack 5

## Развертывание проекта
1. Скачать проект с репозитория
2. Выполнить команду
   ```
   npm install
   ```
3. Для запуска в режиме разработки
   ```
   npm run serve
   ```
3. Для запуска в режиме продакшена
   ```
   npm run build
   ```
   
## Стек
- PUG
- SASS (SCSS)
- POSTCSS
- Babel
- ES6+ syntax
